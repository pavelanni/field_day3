Two Morse Code (CW) Stations

These two stations are located at the North-East corner of the parking lot. The stations are working exclusively using Morse Code mode (aka CW). You can listen to the exchange, but please don't disturb the operators.

CW Stations captain is RJ WF4W. Please contact him if you have any questions.

Phone Station

This station is located at the South-West corner of the parking lot. It mostly uses Phone mode (a.k.a. SSB or single-side band).

Satellite Station

This station is located at the North-East corner of the parking lot. It uses Amateur Radio satellites for contacts. You can watch the map and see when the next satellite will pass our location.

Station captains are Lori K4UPI and Daryl K4RGK. Please contact him if you have any questions.

Get-on-the-Air (GOTA) Station

This station is located at the South-East corner of the parking lot, next to the sign-in booth. It uses Phone mode and if you've never been on the air before, you can make several contacts there to get a "taste" of Amateur Radio.
