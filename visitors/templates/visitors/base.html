Morse Code (CW) Station

This station is located at the North-East corner of the parking lot. The station is working exclusively using Morse Code mode (a.k.a. CW). You can listen to the exchange, but please don't disturb the operators.

CW Station captain is Don KK4E. Please contact him if you have any questions.

Phone Station

This station is located at the South-West corner of the soccer field. It mostly uses Phone mode (a.k.a. SSB or single-side band). The station captain is Mark KJ4YM

Swing Station

This station is located at the North-East corner of the parking lot. It uses all modes: Phone, CW, Digital (FT8). The station captain is Wes W3WL

Satellite Station

This station is located at the North-East corner of the parking lot. It uses Amateur Radio satellites for contacts. You can watch the map and see when the next satellite will pass our location.

Station captains are Lori K4UPI and Daryl K4RGK. Please contact them if you have any questions.

6 meter Station

This station is located at the North-East corner of the parking lot. It uses "the magic band" - 6 meter band. If propagation is good this band can bring great surprises.

Station captain is John K4SQC . Please contact him if you have any questions.

Get-on-the-Air (GOTA) Station

This station is located at the South-East corner of the parking lot, next to the sign-in booth. It uses Phone mode and if you've never been on the air before, you can make several contacts there to get a "taste" of Amateur Radio. Station captain is Terry W4YBV .
